
Exercício 1.1 Quais são os 3 tipos de códigos básicos de uma página de hipertexto? Explique para que serve cada um dos tipos.

R: Os três tipos básicos são HTML, CSS e JavaScript. html define a estrutura e o conteúdo da página web. CSS: Estiliza a página, definindo o layout e os aspectos visuais. JavaScript: Torna a página interativa e dinâmica, permitindo manipulação de conteúdo e interação com o usuário.

Exercício 1.2 Quantos arquivos CSS e Javascript podem estar vinculados a uma mesma página de hipertexto? Justifique sua resposta.

R: Não há um limite específico, você pode vincular quantos arquivos forem necessários para estilizar e adicionar funcionalidades à página. Separar esses arqvuis diferentes pode ajudar a melhorar a organização do site. 

Exercício 1.3 É possível diferentes páginas de hipertexto terem vínculo com os mesmos arquivos Javascript e CSS? Justifique sua resposta.

R:Sim, é possível, e é uma prática comum para reutilizar estilos e funcionalidades em diferentes páginas.

Exercício 1.4 Descreva a diferença entre uma página de hipertexto estática e uma dinâmica, pontuando as vantagens e desvantagens de cada um dos dois tipos.

R: Página estática: Conteúdo fixo, fácil cacheamento e baixo custo de hospedagem.
Página dinâmica: Personalização dinâmica, interatividade avançada, mas complexa de desenvolver.

Exercício 1.5 Qual é o caminho percorrido por uma requisição HTTP a uma página de hipertexto estática?

R: Quando o usuário digita a URL da aplicação desejada, ocorre uma conversão do DNS para o IP correspondente. Em seguida, uma requisição HTTP é enviada para o servidor. Após receber a requisição corretamente, o servidor pode retornar um arquivo HTML estático ou realizar processamento adicional, como acessar ou modificar dados no banco de dados. Após esse processamento, uma resposta é enviada do servidor para o cliente, também via HTTP.
